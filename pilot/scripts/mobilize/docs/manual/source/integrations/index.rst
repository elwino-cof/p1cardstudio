Integrations for mobilize.js
=======================================

This chapter shows how to integrate mobilize.js to different systems.

mobilize.js uses a concept of *extender*. Extenders are the adaption of mobilize.js 
for a specific web site or content
management system. Here are integration instructions for some of the common
systems mobilize.js supports out of the box.

Integrations also show web server specific tricks and other tricks
you might need to be aware of.
